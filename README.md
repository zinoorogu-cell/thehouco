# The HOU Co. — Professional Website

## Project Overview

A complete, multi-page professional website for **The HOU Co.**, a premium business consulting firm founded by **Hassan Olawunmi Uthman** that helps Nigerians in the diaspora build and grow successful businesses in Nigeria.

---

## Pages & Features

### ✅ Completed Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero, stats, about strip, services preview, clients, fear section, gallery, CTA |
| About | `about.html` | Founder story, company purpose, values, fears, clients, CTA |
| Services | `services.html` | Detailed breakdowns of all 5 services with images |
| Rates | `rates.html` | All pricing cards, FAQ accordion, value proposition |
| Contact | `contact.html` | Contact form, office locations, contact details |

### ✅ Functional Entry URIs

| Path | Description |
|------|-------------|
| `index.html` | Main landing page |
| `about.html` | About Hassan & The HOU Co. |
| `services.html` | All services |
| `services.html#diaspora` | Diaspora Business Building |
| `services.html#consulting` | One on One Consulting |
| `services.html#marketing` | Marketing & Growth Advisory |
| `services.html#leadership` | Leadership & Culture Consulting |
| `services.html#speaking` | Speaking Engagements |
| `rates.html` | Pricing |
| `contact.html` | Contact / Enquiry form |
